# cgzd_pay

> 支付领域组件（Maven 多模块工程）：沉淀支付领域模型、通用工具与对外调用 SDK（Feign）。

## 项目简介

`cgzd_pay` 是一个面向“支付/收单/退款/对账”等业务的领域组件化工程，采用 Maven 多模块拆分，将：
- **通用能力**（常量、工具、枚举、统一异常/返回结构等）
- **领域模型**（聚合根、值对象、领域服务、领域事件等）
- **对外调用 SDK**（Feign Client、DTO、拦截器/签名/鉴权等）

进行分层沉淀，便于在不同业务系统中复用与集成。

> 说明：本仓库为通用能力与工程结构展示，不包含任何敏感配置与生产环境密钥。

---

## 模块结构

仓库为 Maven 多模块工程，根目录 `pom.xml` 聚合以下模块：  
- `cgzd-common-pay`：通用基础能力（常量/枚举/工具类/统一异常等）  
- `cgzd-domain-pay`：支付领域模型与领域服务（领域对象、状态流转、校验规则等）  
- `cgzd-feign-pay`：对外调用 SDK（Feign Client、请求/响应 DTO、签名鉴权、拦截器等）

> 你可以根据实际代码把“模块职责”再细化成 3~6 个 bullet（面试加分）。

---

## 技术栈

- Java：8+
- Maven：多模块聚合构建
- Feign：对外 HTTP/服务调用（如有）
- Spring Boot / Spring Cloud
